# ch02-quick-start

claude-mem 安装验证与推荐配置模板。

## 运行

```bash
npm install
npm run demo    # 验证安装
npm run config  # 输出推荐配置
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `src/verify-install.ts` | 检查 claude-mem 是否安装正确（目录、端口） |
| `src/config-templates.ts` | 输出推荐的 settings.json 配置模板 |

更多讨论见 [inferloop.dev](https://inferloop.dev)
